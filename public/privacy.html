Política de Privacidade

Última atualização: 10/07/2025

1. Dados Coletados

Este aplicativo não coleta, armazena ou compartilha dados pessoais dos usuários. Utilizamos apenas tokens temporários fornecidos pela API oficial do TikTok.

2. Uso de Informações

Todos os dados processados são:

  • Exclusivos para operações de automação permitidas
  • Apagados imediatamente após o processamento
  • Nunca repassados a terceiros

3. Segurança

Implementamos medidas técnicas para garantir a segurança das interações com a API do TikTok, incluindo:

  1. Criptografia de todas as requisições
  2. Armazenamento local de tokens sem dados pessoais
  3. Revisões periódicas de segurança
